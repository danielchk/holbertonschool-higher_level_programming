=============================
The ``1-my_list`` module
=============================
Using ``MyList``
---------------------
1. Empty list
    >>> my_list = MyList()
    >>> my_list
    []
    >>> my_list.print_sorted()
    []
2. List positive
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
3. List, one negative
    >>> my_list.append(-5)
    >>> my_list
    [1, 4, 2, 3, 5, -5]
    >>> my_list.print_sorted()
    [-5, 1, 2, 3, 4, 5]
